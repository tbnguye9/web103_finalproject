## Reflection

### 1. What went well during this unit?

Our team successfully set up the GitHub project management tools including issues, project board, and milestones. We were also able to begin development of the frontend using React and Vite, and implemented basic functionality such as creating and displaying tasks.

### 2. What were some challenges your group faced in this unit?

One challenge we faced was understanding how to properly set up GitHub project boards and assign tasks to team members. We also had some confusion when setting up the development environment and ensuring everyone had access to the repository.

### 3.Did you finish all of your tasks in your sprint plan for this week? If you did not finish all of the planned tasks, how would you prioritize the remaining tasks on your list?

We completed the core tasks for this milestone, including setting up the project structure and implementing basic task creation and display features. Some advanced features were not completed and will be prioritized in the next sprint.
